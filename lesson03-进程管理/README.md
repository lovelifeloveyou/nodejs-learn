# lesson03-进程管理

## 本章节涉及知识点

* `child_process`模块。`Node` 通过`child_process`模块提供了类似`popen(3)`的处理三向数据流（stdin/stdout/stderr）的功能。它能够以完全非阻塞的方式与子进程的`stdin`、`stdout`和`stderr`以流式传递数据。
